# yuval/reference/

תיקיית **חומרי השראה** של יובל - מעצב התמונות.

## מטרה

כאן יושבות תמונות / טקסטים / מסמכים שמגדירים את הסגנון הויזואלי של הצוות. יובל סורק את התיקייה הזו לפני כל יצירת תמונה כדי לשמור על **עקביות ויזואלית** בין כל התוצרים.

## מה לשים כאן

- תמונות לדוגמה (PNG / JPG) שמייצגות את הסגנון הרצוי
- קובצי `.md` עם תיאור הסגנון (פלטת צבעים, אווירה, אלמנטים חוזרים)
- מסמכי brand guidelines אם יש

## הערות

- אם התיקייה ריקה - יובל יעבוד בשיקול דעת עצמאי ויציין זאת בדיווח לראובן.
- יובל **לא משנה** את הקבצים בתיקייה הזו - היא קלט בלבד.
- התוצרים שלו נשמרים ב-`yuval/outputs/`, לא כאן.
